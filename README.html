Getting Started
  1. Invite Disconnect
  2. Go to a channel Disconnect can read and send messages in
  3. Type something below to get its held information
    • !help - Get information basic information and help with Disconnect.
    • !about - Get information about Disconnect.
    • !command-list - Get all the commands in Disconnect sent to your DM's.
    • !bot-stats - Get all the stats of the Disconnect.
    • !command-help - Use this command to get help with a command.
